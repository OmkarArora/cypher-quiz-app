# Cypher Quiz

Quiz App built with TypeScript in React.

Live link: https://cypher-quiz.netlify.app

Backend: https://github.com/OmkarArora/cypher-backend

---

## Tech stack

- Frontend
  - React
  - TypeScript
  - React Router
- Backend
  - Express
  - Mongoose
  - MongoDB
